# Yunning You - Personal Homepage

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fyouyunning%2Fyyn-homePage)

Built with **Next.js 16 + Tailwind CSS 4 + TypeScript**.

## Demo

- Portfolio: [youyunning.github.io/yyn-homePage](https://youyunning.github.io/yyn-homePage)

## Features

- ⚙️ **Quick setup:** Get started in minutes by editing the config files.
- 📰 **Rich portfolio:** Showcase news, projects, experiences, and more.
- 📝 **Blog support:** Write effortlessly with Markdown.
- 🌍 **Internationalization:** English & Chinese I18n out of the box.
- 🌗 **Light & dark mode:** Toggle color themes.
- 🚀 **Vercel ready:** Deploy in seconds.
- 📱 **Responsive design:** Looks great on all devices.
- 🤖 **SEO optimized:** Metadata, JSON-LD, Open Graph tags.
- 📊 **Analytics ready:** Google Analytics 4, GTM, Search Console.

**Built with:**
- ⚡ Next.js 16 (App Router)
- 🔥 TypeScript
- 💎 Tailwind CSS 4
- 💅 shadcn/ui & magic ui
- 🎬 Framer Motion
- 💯 Lighthouse optimized

## Getting Started 🚀

### 1. Clone & install

```bash
git clone https://github.com/youyunning/yyn-homePage
cd yyn-homePage
pnpm install
```

### 2. Launch dev server

```bash
pnpm dev
```

Visit [`http://localhost:3000`](http://localhost:3000).

### 3. Customize

- Edit main info in `src/data.tsx` and `src/data/site.ts`
- Edit text & content in `src/i18n/messages/`
- Add blog posts in `content/blog/`

### 🚀 Deploy to Vercel

Click the Deploy button above, or connect this repo to Vercel directly.
